Historical Weather Data API

Temperature in date on station

URL format: http://127.0.0.1:5000/api/v1/station_STAID/date

URL Example for one station for one date: http://127.0.0.1:5000/api/v1/10/19881025


All temperature measurements on station

URL format: http://127.0.0.1:5000/api/v1/station_STAID/

URL Example for one station: http://127.0.0.1:5000/api/v1/10


All temperature measurements on station in specific year

URL format: http://127.0.0.1:5000/api/v1/annual/station_STAID/year

URL Example for one station for one year: http://127.0.0.1:5000/api/v1/annual/10/1992


Chart of temperature measurements on station in specific year

URL format: http://127.0.0.1:5000/visualization/station_STAID/year

URL Example for one station for one year: http://127.0.0.1:5000/visualization/10/1992
